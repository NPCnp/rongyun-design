# 榕韵亲子定制 - 部署指南

## 📋 部署前准备

### 1. 确认 Supabase 配置

请先在 Supabase 后台确认以下信息：

**项目 URL**（在 Settings → API 中）：
- 格式类似：`https://xxxxx.supabase.co`

**匿名访问密钥**（anon/public）：
- 格式类似：`sb_xxxxxx_xxxxxxxx`

### 2. 更新配置文件

在以下文件中填入正确的 Supabase 信息：

#### 文件1：`src/config.js`
```javascript
var SUPABASE_URL = 'https://你的项目ID.supabase.co';
var SUPABASE_ANON_KEY = '你的anon密钥';
```

#### 文件2：`admin.html`
```javascript
const SUPABASE_URL = 'https://你的项目ID.supabase.co';
const SUPABASE_ANON_KEY = '你的anon密钥';
```

## 🚀 方案一：Vercel 部署（推荐）

### 步骤1：准备 GitHub 仓库

1. 登录 GitHub
2. 创建一个新仓库（命名任意，如 `rongyun-diy`）
3. 选择 Public 或 Private 都可以
4. 点击 "Create repository"

### 步骤2：上传文件

**方式A：网页上传（简单）**
1. 在仓库页面点击 "uploading an existing file"
2. 把整个 DIY 文件夹的文件拖进去
3. 填写提交信息，点击 "Commit changes"

**方式B：Git 命令行**
```bash
cd c:\Users\15759\Desktop\DIY
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

### 步骤3：在 Vercel 部署

1. 访问 [vercel.com](https://vercel.com)
2. 用 GitHub 账号登录（Sign Up → Continue with GitHub）
3. 点击 "Add New Project"
4. 选择您的 GitHub 仓库
5. 点击 "Import"
6. 确认配置（默认即可），点击 "Deploy"
7. 等待 1-2 分钟，部署完成！

### 步骤4：获取访问链接

部署完成后，Vercel 会给您一个链接，类似：
- `https://rongyun-diy.vercel.app`

游客可以直接用手机访问这个链接！

## 🚀 方案二：Netlify 部署（最简单）

### 步骤：

1. 访问 [netlify.com](https://netlify.com)
2. 注册/登录账号
3. 点击 "Add new site" → "Deploy manually"
4. 直接把整个 DIY 文件夹拖到上传区域
5. 等待几秒钟，部署完成！

您会得到一个免费链接，例如：
- `https://magical-unicorn-12345.netlify.app`

---

## 🚀 方案三：GitHub Pages（免费）

### 步骤：

1. 创建 GitHub 仓库并上传文件（同方案一步骤1-2）
2. 进入仓库 Settings
3. 找到 "Pages" 部分
4. 在 "Branch" 下拉菜单选择 `main`（或 `master`）
5. 点击 "Save"
6. 等待 2-5 分钟，页面会显示访问链接

访问链接类似：
- `https://你的用户名.github.io/仓库名/`

## 📱 使用说明

### 给游客使用

1. 把部署后的链接（如 `https://rongyun-diy.vercel.app`）生成二维码
2. 打印出来贴在门店
3. 游客扫码即可开始设计和下单

### 管理端使用

1. 访问 `https://你的链接/admin.html`
2. 可以查看、管理所有订单
3. 订单会自动从 Supabase 同步

## 🔧 常见问题

### Q: Supabase 连接失败？
A: 检查以下几点：
1. URL 和密钥是否正确
2. 数据库表是否已创建
3. 行级别安全策略是否允许公开读写

### Q: 手机访问很慢？
A: 建议使用 Vercel 或 Netlify，它们有全球 CDN 加速

### Q: 想换域名？
A: Vercel 和 Netlify 都支持绑定自定义域名
