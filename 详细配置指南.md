# 📖 超详细Supabase配置指南

## 第一步：注册并登录

### 1.1 打开网站
在浏览器中打开：https://supabase.com

### 1.2 点击开始
点击页面右上角的 **"Start your project"** 按钮（绿色的）

### 1.3 登录
- 如果你有GitHub账号，点击 **"Continue with GitHub"** 登录（推荐）
- 或者用邮箱注册

---

## 第二步：创建新项目

### 2.1 点击新建项目
登录后会自动跳转到Dashboard，点击 **"New Project"** 按钮

### 2.2 填写项目信息
按下面的样子填写：

| 项目 | 填写内容 |
|------|---------|
| **Name** | `rongyun-diy` (随便填一个你喜欢的名字) |
| **Database Password** | 设置一个密码，**一定要记住！** |
| **Region** | 选择亚太地区：`Singapore` (新加坡) 或 `Tokyo` (日本) |
| **Pricing Plan** | 选免费版 `Free` |

### 2.3 点击创建
点击 **"Create new project"** 按钮

### 2.4 等待创建
现在需要等待 **2-5分钟**，看到页面显示 "Setting up your database" 或类似提示，请耐心等待。

---

## 第三步：创建订单表

### 3.1 打开SQL编辑器
数据库准备好后，左侧菜单点击 **"SQL Editor"**

### 3.2 新建查询
点击 **"New Query"** 按钮

### 3.3 复制粘贴代码
把下面的代码完整复制进去：

```sql
-- 创建订单表
CREATE TABLE orders (
    id bigserial PRIMARY KEY,
    order_no bigint UNIQUE NOT NULL,
    name text NOT NULL,
    phone text NOT NULL,
    pickup_code text,
    design text,
    status text DEFAULT '待制作',
    time text,
    created_at timestamp DEFAULT now()
);

-- 允许任何人读取订单
ALTER TABLE orders ENABLE ROW LEVEL SECURITY;
CREATE POLICY "允许所有人读取" ON orders FOR SELECT USING (true);

-- 允许任何人插入订单
CREATE POLICY "允许所有人插入" ON orders FOR INSERT WITH CHECK (true);

-- 允许任何人更新
CREATE POLICY "允许所有人更新" ON orders FOR UPDATE USING (true);

-- 允许任何人删除
CREATE POLICY "允许所有人删除" ON orders FOR DELETE USING (true);
```

### 3.4 运行代码
点击右下角的 **"Run"** 按钮执行

### 3.5 验证成功
看到 "Success. No rows returned" 就表示成功了！

---

## 第四步：获取连接信息

### 4.1 打开设置
左侧菜单点击底部的 **"Settings"**（齿轮图标），然后选择 **"API"**

### 4.2 复制两个信息
你会看到两个重要信息，请复制保存：

1. **Project URL**：
   - 类似这样：`https://xxxxxxx.supabase.co`
   - 点击复制按钮复制

2. **anon public**：
   - 一长串字符，以 `eyJ` 开头
   - 点击复制按钮复制

**⚠️ 重要提示**：这两个信息要保管好，不要泄露给别人！

---

## 第五步：配置设计页面

### 5.1 打开设计页面
在浏览器中打开：
`https://npcnp.github.io/rongyun-design/standalone/`

### 5.2 点击配置
点击页面顶部的 **"⚙️ 配置云端"** 按钮

### 5.3 填写信息
- **Project URL**：粘贴刚才复制的第一个地址
- **anon public key**：粘贴刚才复制的第二个长字符串

### 5.4 保存
点击 **"保存配置"** 按钮

### 5.5 验证成功
看到提示 "配置成功！" 就OK了！

---

## 第六步：配置管理后台

### 6.1 打开后台
在电脑上找到 `admin.html` 文件，双击打开

### 6.2 填写配置
在页面顶部配置区域：
- Project URL：粘贴同样的地址
- anon public key：粘贴同样的key

### 6.3 连接
点击 **"保存配置并连接"**

### 6.4 验证成功
看到提示 "Supabase 连接成功！" 就完成了！

## 常见问题

### Q: 配置后还是看不到订单？
A: 检查以下几点：
- 确保 Project URL 和 Key 复制完整了
- 确认 SQL 代码执行成功
- 刷新一下后台页面

### Q: 手机端需要每次配置吗？
A: 不需要！配置一次会自动保存，以后都能用

### Q: 免费版够用吗？
A: 完全够用！每天5万次请求，500MB数据库

### Q: 换手机了怎么办？
A: 在新手机上重新配置一次就行
