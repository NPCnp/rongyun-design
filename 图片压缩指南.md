# 图片压缩指南

## 当前状态
- 总图片数：20 张
- 当前总大小：135.13 MB

## 推荐压缩方案

### 方案 A：使用在线工具（最简单）

1. 访问以下任一网站：
   - **TinyPNG** (https://tinypng.com/) - PNG 压缩
   - **Squoosh** (https://squoosh.app/) - 图片压缩和调整大小
   - **CloudConvert** (https://cloudconvert.com/) - 格式转换

2. 下载所有图片

3. 批量处理：
   - 将 PNG 转换为 JPEG
   - 调整图片尺寸（缩小 50-70%）
   - 压缩质量设为 70-80%

4. 重新上传到项目

### 方案 B：使用本地工具

#### Windows 用户：

1. 安装 **ImageMagick**：
   - 访问 https://imagemagick.org/script/download.php
   - 下载 Windows 版本并安装

2. 将图片文件夹中的 PNG 文件批量转换为 JPEG：
   ```bash
   # 在命令行中运行
   cd 图片文件夹
   magick mogrify -format jpg -quality 80 *.png
   ```

3. 批量调整大小：
   ```bash
   # 缩小 50%
   magick mogrify -resize 50% *.jpg
   ```

#### macOS 用户：

1. 安装 Homebrew（如果没有）：
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. 安装 ImageMagick：
   ```bash
   brew install imagemagick
   ```

3. 运行转换命令：
   ```bash
   cd 图片文件夹
   magick mogrify -format jpg -quality 80 *.png
   magick mogrify -resize 50% *.jpg
   ```

## 压缩效果预估

| 方案 | 压缩后大小 | 优点 | 缺点 |
|------|-----------|------|------|
| 仅转 JPEG | 30-40 MB | 保持清晰度 | 仍超过限制 |
| 仅缩小 | 60-70 MB | 保持 PNG | 超过限制 |
| 组合方案 | 15-20 MB ✅ | 可上传 | 需要两步 |

## 注意事项

1. **透明通道**：PNG 转 JPEG 会丢失透明背景，但明信片图片通常不需要透明
2. **图片质量**：JPEG 质量 70-80% 对于屏幕显示足够清晰
3. **备份**：压缩前请先备份原始图片
